How to develop the project
=============================================
Preparation for development this project:
            
1) Install maven and prepare the libraries
   Step 1: Install libraries from folder ./lib into your local maven repository by execute script install.bat         

How to use code
=============================================
Step 1) Execute test case to clone project from BitBucket to local
View sample test case: /git-wrapper/src/test/java/m/k/s/gitwrapper/GitServiceTests.java

Method: public void testClone()

Step 2) Execute test case to write all logs into Excel file
View sample test case: /git-wrapper/src/test/java/m/k/s/gitwrapper/GitServiceTests.java

Method: public void testGetInfo()
